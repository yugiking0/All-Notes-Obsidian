# Azure

## 1. Azure là gì?
- Microsoft Azure là một nền tảng điện toán đám mây cung cấp một số ứng dụng và dịch vụ cho doanh nghiệp. Chúng bao gồm các dịch vụ trong danh mục sau:
	-   Containers
	-   Compute
	-   Machine learning
	-   Developer tools
	-   IoT
